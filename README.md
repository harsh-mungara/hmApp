# hmApp
# Project setup
- ### Clone the repo  
        git clone https://github.com/harsh-mungara/hmApp.git
- ### checkout dev branch (development)
        git checkout dev
- ### install modules
        npm install
- ### Run on android : npm run android
- ### Before run on ios fire below command
        cd ./ios && pod install &&  cd ..
